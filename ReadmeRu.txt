Command line arguments
==========================================

/noscan - не искать пароли при старте
/plugins - использовать плагины
/nodrv - не использовать драйвер режима ядра для чтения заблокированных файлов
/newlang - программа создаст en.lang файл, который можно перевести на альтернативный язык и получить ключ бесплатно
/extlang - использовать внешний (ext.lang) файл, используется для проверки сделанных переводов
/export - автоматически сохранить пароли во внешний файл
